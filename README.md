# RidgeMMD-for-Single-Cell-Perturbation-Prediction
RidgeMMD is a distribution-aware computational framework designed to predict high-dimensional genome-wide transcriptomic responses (10,000 genes) from a minimal set of 200 Landmark-like highly variable genes (HVGs). While standard regression models often suffer from "mean collapse"—a phenomenon where predictions gravitate toward the population average and lose cellular heterogeneity—RidgeMMD integrates Maximum Mean Discrepancy (MMD) as a loss function constraint to align the predicted manifold with the ground-truth single-cell distribution in a Reproducing Kernel Hilbert Space (RKHS).  Validated on the Tahoe-100M dataset. By precisely capturing synergistic expression patterns in critical oncogenic pathways such as Apoptosis and STAT3 signaling, RidgeMMD provides an efficient and scalable solution for virtual compound screening and large-scale perturbation analysis.
